# מפרטי מסכים - Hoogi UI Sandbox

## סקירה כללית

מסמך זה מספק מפרטים מלאים לכל המסכים הזמינים ב-UI Sandbox, כולל מבנה, מצבים שונים וכללי רספונסיביות.

## רשימת מסכים

### 1. לוח בקרה ראשי (Dashboard)
**קובץ**: `screens/dashboard.html`  
**תיאור**: מסך הבית הראשי עם סקירה כללית של הנתונים

#### מבנה המסך:
```
┌─────────────────────────────────────────┐
│ Header (Logo + User Menu)               │
├─────────────────────────────────────────┤
│ Welcome Section                         │
│ - כותרת עם שם משתמש                     │
│ - כפתור "שאלון חדש"                    │
├─────────────────────────────────────────┤
│ Top Stats Cards (2x1)                   │
│ - תגובות (Responses)                    │
│ - לידים חמים (Hot Leads)                │
├─────────────────────────────────────────┤
│ Bottom Stats Cards (4x1)                │
│ - הכנסות היום                          │
│ - השלמנו                               │
│ - בטיפול                               │
│ - חדשים                                │
├─────────────────────────────────────────┤
│ Questionnaires Section                  │
│ - רשימת שאלונים                        │
│ - כרטיס לכל שאלון                      │
├─────────────────────────────────────────┤
│ Quick Actions (3x1)                     │
│ - יצירת תוכן                           │
│ - ניתוח נתונים                         │
│ - הגדרות                               │
└─────────────────────────────────────────┘
```

#### מצבים:
- **רגיל**: מצב ברירת מחדל עם נתונים
- **ריק**: ללא שאלונים (מציג כרטיס "צור שאלון חדש")
- **טעינה**: מציג סקלטונים במקום נתונים
- **שגיאה**: מציג הודעת שגיאה במקום נתונים

#### רספונסיביות:
- **Mobile (sm)**: עמודה אחת, כרטיסים במלוא הרוחב
- **Tablet (md)**: 2 עמודות לסטטיסטיקות, 2 לפעולות מהירות
- **Desktop (lg)**: 4 עמודות לסטטיסטיקות, 3 לפעולות מהירות

#### רכיבים מרכזיים:
- Header עם לוגו ותפריט משתמש
- כרטיסי סטטיסטיקה עם אייקונים
- רשימת שאלונים עם מידע מפורט
- כפתורי פעולה מהירה

## כללי רספונסיביות

### Breakpoints:
- **sm**: 640px ומעלה
- **md**: 768px ומעלה  
- **lg**: 1024px ומעלה

### עקרונות:
1. **Mobile First**: עיצוב מתחיל ממובייל
2. **Progressive Enhancement**: שיפור הדרגתי
3. **Touch Friendly**: כפתורים גדולים במובייל
4. **Content Priority**: תוכן חשוב ראשון

### דפוסים:
- **Grid to Stack**: רשת הופכת לעמודה במובייל
- **Hide on Mobile**: אלמנטים לא חיוניים מוסתרים
- **Horizontal Scroll**: טבלאות עם scroll אופקי
- **Card Layout**: טבלאות הופכות לכרטיסים
